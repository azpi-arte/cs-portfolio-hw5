Frontend Development – Splagen Nonprofit

Description

Contributed to Splagen’s platform by implementing frontend features within a MERN ecosystem, enhancing UI workflows, and improving user engagement for nonprofit-facing tools.

Role

Built TypeScript-driven components, integrated REST endpoints, and collaborated with backend engineers to refine data-heavy views used by volunteers and project coordinators.

    Tech Utilized

  • TypeScript
  • React
  • Node.js
  • Express
  • MongoDB
  • Agile/Scrum

    Challenges

  1. Component Architecture

    Created modular and scalable UI components for dashboards and form flows, ensuring predictable state management and alignment with team standards.

  2. API Integration

    Connected React views to Splagen’s backend, handling validation, error states, and loading patterns to streamline user workflows.

  3. Agile Collaboration

    Participated in two-week sprints, coordinated through stand-ups and backlog reviews, and supported teammates by maintaining clear documentation.

  4. Accessibility & UI Polish

    Incorporated accessible markup patterns and refined layouts to accommodate Splagen’s diverse volunteer base.
